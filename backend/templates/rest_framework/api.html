Eine maschinenlesbare Spezifikation im OpenAPI v3 Format lassen sich unter /openapi.json abrufen. Damit lassen sich schnell und einfach eigene Clients in vielen Programmiersprachen generieren.

Alternativ sind wir auch auf Twitter oder per Mail zu erreichen.

Sowohl API als auch die App lassen sich auch auf eigenen Servern hosten. Hinweise dazu finden sich ebenfalls im Repository codeformuenster/muenster-jetzt auf GitHub.
